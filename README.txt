Evans Revision Lab v6 - Mistake Book Upgrade

What changed:
1. Added Review My Mistakes button on the home screen.
2. Wrong answers are saved automatically in the browser's localStorage.
3. Saved records include question, wrong answer, correct answer, topic, subject, mistake type, status, times wrong, and correct streak.
4. Added Mistake Review mode.
5. If Evans answers a saved mistake correctly twice in review mode, it becomes Mastered.
6. Added Weak Topics & Progress page.

Important:
- Mistakes are stored locally in the same browser/device.
- If the browser cache/localStorage is cleared, the saved mistake book will reset.
- Upload index.html, style.css and script.js to the website root.
